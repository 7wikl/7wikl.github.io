在html中，如果想要一个块元素一直保持在最底部，可以采取如下方法

``` <!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title></title>
		<style type="text/css">
			.foot{
				position: absolute;
				width:99%;
				text-align: center;
				bottom: 0px;
			}
		</style>
	</head>
	<body>
		<div class="head">3</div>
		<div class="mid">2</div>
		<div class="foot">1</div>
	</body>
</html>

```
如果页面还有下拉页面的话，可以尝试把bottom改为负值

``` bottom: -60px;
```
